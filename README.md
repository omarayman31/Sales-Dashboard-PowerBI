# 📊 Sales Dashboard - Power BI

## 📌 Project Overview

This project demonstrates a complete end-to-end sales analysis workflow using Python and Power BI. The raw sales data was cleaned and transformed using Python before being imported into Power BI for data modeling, KPI creation, and dashboard development.

---

## 🔄 Project Workflow

### 1. Data Collection
- Imported the raw sales dataset.

### 2. Data Cleaning (Python)
- Removed duplicate records.
- Handled missing values.
- Converted data types.
- Created new calculated columns.
- Exported the cleaned dataset.

### 3. Data Modeling (Power BI)
- Built a Star Schema.
- Created relationships between fact and dimension tables.

### 4. DAX Measures
- Total Sales
- Total Profit
- Total Orders
- Average Order Value (AOV)
- Customer Lifetime Value (CLV)

### 5. Dashboard Design
- KPI Cards
- Sales Trends
- Category Analysis
- Customer Analysis
- Interactive Filters (Slicers)
